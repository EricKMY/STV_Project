# How To Run The Test  
1.Install [appium](http://appium.io/)<br>
2.Install [Android Studio](https://developer.android.com/studio/?gclid=Cj0KCQjw6PD3BRDPARIsAN8pHuHK5oztDInd2zIiI2gZFu3UESvaMolx2u5OPk9tThK0rkGyrMmDJBoaApRnEALw_wcB&gclsrc=aw.ds)<br>
3.Install [appiumlibrary](https://pypi.org/project/robotframework-appiumlibrary/)<br>
4.Install [RED](https://github.com/nokia/RED)<-optional, may run by cmd.<br>
5.Create apk of "moneywallet" through "Android Studio"<br>
6.Create a Virtual Device through Android Studio and run  <br>
* AVD Name: Pixel 2 API 25<br>
* Nougat: Android 7.1.1 x86<br>

7-1.Import and run "moneywallet_test_automation" through RED IDE<br>
7-2.Run by cmd<br>
> python -m robot "."
